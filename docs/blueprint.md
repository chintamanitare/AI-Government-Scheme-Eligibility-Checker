# **App Name**: Aadhar Assist AI

## Core Features:

- Eligibility Check: Allows users to input their details (age, income, state, etc.) to check their eligibility for government schemes.
- AI-Powered Scheme Advisor: Utilizes Gemini AI to analyze user data, identify relevant schemes, determine eligibility, and rank schemes by relevance.
- Detailed Scheme Information: Provides clear, citizen-friendly explanations of each scheme, including eligibility status, priority ranking, and required documents.
- Multilingual Support: Offers results and explanations in the user's preferred language (English, Hindi, Marathi).
- Document Checklist: Generates a checklist of required documents for each eligible scheme, powered by an AI tool to derive a relevant and complete list from the latest updates to governmental schemes.
- Step-by-Step Application Guide: Provides a step-by-step guide for applying to each eligible scheme, derived by an AI tool that reasons when to provide specific information, using online information to augment a proprietary knowledge base.
- Firestore Data Storage: Saves user eligibility checks to Firestore for analytics and offline result recovery.

## Style Guidelines:

- Primary color: Saffron (#FF9933), reflecting the Indian government theme and promoting trust.
- Background color: Light beige (#F5F5DC), a very lightly desaturated tint of the primary saffron to offer a neutral background and create contrast with the other elements.
- Accent color: Forest Green (#228B22), another color in the Indian flag, is a highly saturated counterpart that adds contrast and aids user focus
- Body and headline font: 'PT Sans' (sans-serif) combines a modern look and a little warmth, making the application user-friendly for all citizens.
- Use relevant icons for different categories and schemes to make the app more visually appealing.
- Implement a mobile-first responsive layout for accessibility across various devices.
- Incorporate subtle animations for loading states and transitions to enhance user experience.
